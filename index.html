MAW: Modular Audio for the Web

This project is an experiment in using the Web Audio API to create a dynamic audio graph with different types of sound sources and processing nodes can be wired together on-the-fly.

Run the demo (NOTE: it probably only works in recent versions of Chrome)

Check out the source code

Click the amplifier to start the audio. Each row let's you select a node type to insert into the audio graph. On the first row, you select a sound source (a wavetable oscillator). The second row is distortion. The third row is filters.
