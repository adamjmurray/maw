This project is an experiment in using the Web Audio API to create a dynamic audio graph with different types of sound sources and processing nodes can be wired together on-the-fly.

Run the demo

For more info on the Web Audio API, see http://dvcs.w3.org/hg/audio/raw-file/tip/webaudio/specification.html http://chromium.googlecode.com/svn/trunk/samples/audio/index.html http://www.html5rocks.com/en/tutorials/webaudio/intro
